- 自动排序引入的顺序
  - mac: `option` + `shift` + `o`
  - windows: `control` + `shift` + `o`

- 自动排序`JSON`文档
  - mac: `command` + `shift` + `p` --> `JSON: Sort Document`

- 重启`typescript`服务
  - mac: `command` + `shift` + `p` --> `TypeScript: Restart TS Server`
