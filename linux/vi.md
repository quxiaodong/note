#### 命令模式

```bash
# 进入命令模式
vi README.md
```

#### 编辑模式

在编辑模式下，按`Esc`返回命令模式

```bash
# 在命令模式下
# 输入 i 执行插入命令
# 进入编辑模式
```

```bash
# 在命令模式下
# 输入 dd 删除当前行
# 不会进入编辑模式
```

#### 末行模式

```bash
# 在命令模式下
# 输入 /你要搜索的文字
# 再按回车
# 输入 n 向下查找
# 输入 N 向上查找
```

```bash
# 在命令模式下
# 输入 :wq
# 保存并退出
```

```bash
# 在命令模式下
# 输入 :q!
# 退出不保存
```
